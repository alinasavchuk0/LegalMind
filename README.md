# ⚖️ LegalMind AI — Юридичний асистент на базі ШІ

**LegalMind** — це веб-застосунок для автоматизованого аналізу юридичних документів. Проєкт розроблено як демонстрацію можливостей LLM (Large Language Models) у сфері LegalTech.

## 🚀 Основні можливості
- **Миттєвий аудит:** Аналіз тексту договору на наявність прихованих ризиків для орендаря.
- **Експертні поради:** Надання рекомендацій щодо зміни формулювань у документі.
- **Зручний інтерфейс:** Лаконічний дизайн на базі Streamlit.

## 🛠 Технології
- **Backend:** Python 3.x
- **Frontend:** Streamlit
- **AI Core:** Google Gemini API (Model: gemini-2.5-flash)
